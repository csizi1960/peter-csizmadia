This page contains a simple JavaScript line that can kill Netscape 6.0 when you press Reload. If the browser stays alive, then press Reload again. 
window.Packages.java.lang.System.out.println("I'm dead");
There is nothing special in this line, probably any Java command would do it. Notice that Netscape does not die if you do a reload only for this frame (right click-Reload). It dies only if you reload both frames at the same time.
