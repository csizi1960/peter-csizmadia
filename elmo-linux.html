How to make OpenSUSE usable

11.1 (December 2008)

Software repositories. Control Center (YaST) → Software → Software Repositories → Add → ...
URLPackages to install/update
  DVD or http://download.opensuse.org/distribution/11.1/repo/oss krb5-client (for AFS login)
openSUSE-11.1-Oss http://download.opensuse.org/distribution/11.1/repo/oss gsl, gsl-devel
Filesystems http://download.opensuse.org/repositories/filesystems/openSUSE_11.1 openafs-client, openafs-krb5-mit, openafs-kmp-X (AFS driver for your kernel),
Packman ftp://packman.iu-bremen.de/suse/11.1 xmms, MPlayer, libxine1 (MP3, video codecs)
NVIDIA http://download.nvidia.com/opensuse/11.1 nvidia-gfxG02-kmp-default (NVIDIA graphics driver for default kernel)

How to install openafs if it disappeared from the Filesystems repository (it really happens sometimes!)

  1. Install the kernel-source, kernel-syms, krb5-devel and keyutils-devel packages
  2. Get the openafs source rpm:
    http://download.opensuse.org/repositories/filesystems/openSUSE_11.1/src/
  3. Assuming that you use an x86_64 CPU, run the following commands as root: 
    rpmbuild --rebuild openafs-1.4.8*.src.rpm
cd /usr/src/packages/RPMS/x86_64
rpm -Uvh --force openafs-1.4.8*.rpm openafs-client-1.4.8*.rpm openafs-krb5-mit-1.4.8*.rpm openafs-kmp-X.rpm
Installing the Java plugin for 64-bit Firefox
  1. Download and install JRE 1.6 update 12 or later from https://jdk6.dev.java.net/6uNea.html
  2. cd /usr/lib64/browser-plugins
ln -s /usr/java/jdk1.6.0_12/jre/lib/amd64/libnpjp2.so .
